Color Palette#

Colors are such an important part of any project. This framework makes sure they are treated with importance. There are a number of tools and functions allowing for ease of generation and/or manipulation. The tools also ensure apporpiate contrasts for legibility, and proper harmony for overally beauty.

The colors can be pulled in via SCSS $variables and or .text-color classes, depending on the color. The main colors (red, green blue etc) have classes, while their individual color stacks (red-100, red-200, red-300 etc) are referenced via variables.

The colors are generated via a JSON tokens file colors.json found in the tokens/ directory. Once the values are configured to the desired theme, simply run the CLI program to have the new theme and stacks automatically generated and SCSS updated like magic.
